Overview

Author is: Pierre Alletru (pa211@kent.ac.uk)
Supervisor is: Colin Johnson (C.G.Johnson@kent.ac.uk)

Subject of the project is: Kinect Hacking.

Project name of the corpus is: Air.

The Air project is divided in two sub-projects :
- NiFingersGenerator, an OpenNI module to detect fingers. OpenNI is a framework for Natural Interactions (NI). See OpenNI.
- Air, an interface to control Ubuntu and Windows 7 operating systems using hand and fingers gestures.

The archive on the CD is actually a compressed copy of the project repository, available here.

The primary goal of the project was to be able to control an OS with hand and/or fingers gesture.

While the project is not quite stable at this point, the goal of detecting fingers has been reached, as well as the goal of offering some basic gestures controlling the OS. However, gesture recognition often produce errors. The software demonstrate that things are possible, but is currently not usable for confortable real day to day use.

In other words, you will probably encounter some difficulties to correctly manipulate your computer.

Where to start?

1 - The first thing to do is to look at the README file in the sources root directory.

The README contains information about how to install or/and compile the project.

2 - Read the user guide. Available in the 'documentation' directory, or here.

The user guide explains how to use the project.

3 - As it is written in the user guide, watching the video tutorial is probably the best way to learn how to perform the gestures in front of your Kinect.

To get the video, click on the "Downloads" button on this page.
