Detailed Description

Event class. Associate left/right swipe gesture to the action of switching between the current running applications. Inherit from XnVSwipeDetector, since it must detect swipe gestures.

OpenNI callback called when a swipe left or swipe right gesture is recognized. Perform an 'Alt + Tab' keyboard event to switch current applications.

Parameters:

angleThe angle of the movement, with respect to the X axis (horizontal).
